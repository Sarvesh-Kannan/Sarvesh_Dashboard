Key Features

About the Dataset

The dataset contains 1470 records with 35 columns, focusing on demographics, job details, and performance metrics. The primary goal is to analyze factors influencing employee attrition.

Technical Highlights

  • Data preprocessing was carried out using Python and Pandas.
  • EDA (Exploratory Data Analysis) revealed key trends, which were visualized with Power BI.
  • Custom DAX (Data Analysis Expressions) functions provided advanced metrics.

Importance of Preprocessing

Preprocessing ensures clean, consistent, and ready-for-analysis data. By handling missing values and outliers, the accuracy of insights is significantly improved.

Technical Aspects

DAX Functions

  • CALCULATE: Used to modify filter contexts for measures.
  • SUMX: Iterates over a table and sums the values row by row.
  • IF: Implements conditional logic to create dynamic measures.
  • RELATED: Retrieves values from related tables.

Key Performance Indicators

KPIs such as Employee Satisfaction, Attrition Rate, and Training Effectiveness were calculated using DAX. These metrics provide actionable insights for workforce management.
